# ChronoGym 🏋️

A systems design project for a universal gym crowd tracking application.

## Project Overview
ChronoGym helps users find nearby gyms and check real-time crowd levels
to plan their workouts around their schedule.

## Documents Included
- Systems Request Document
- Requirements Definition
- Use Case Diagrams
- Data Flow Diagrams (DFD)
- Entity Relationship Diagram (ERD)
- Interface Structure Design
- Storyboard
